# CyberTamimTech - Personal Portfolio Website

This is a **responsive personal portfolio website** for Abid Al Hasan Tamim, built with simple HTML, CSS, and JavaScript.  
The design is inspired by Facebook's blue and white theme with added features like:

- Dark Mode toggle  
- Typing animation  
- Profile and Cover photo upload (local preview)  
- Skills progress bars  
- Live clock display  
- Sections for Tools, APK Links, Termux Guide, and ESP8266 Projects  
- Animated scroll effects (AOS library)  
- Contact links (Facebook and Email)  

## Features

- Fully responsive for mobile and desktop  
- Easy to customize content and images  
- SEO meta tags for better social sharing previews  
- Simple user interface with smooth animations  

## How to Use

1. Clone or download this repository.  
2. Open `index.html` in any modern browser to view the website locally.  
3. To customize your profile and cover photo, use the upload inputs on the page.  
4. To deploy online, push the code to GitHub and connect your repository with [Vercel](https://vercel.com/) or any static site hosting provider.  

## Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla)  
- [AOS - Animate On Scroll](https://michalsnik.github.io/aos/)
